# 📊 Global Layoffs Data Analysis & Executive Dashboard (2020–2023)

## 📌 Project Overview
This project provides an **Exploratory Data Analysis (EDA)** on global corporate layoffs occurring during and immediately following the COVID-19 pandemic. Using **MySQL**, raw layoff data was cleaned, staged, and analyzed to uncover macro-level hiring trends, industry vulnerabilities, and corporate restructuring patterns across time.
